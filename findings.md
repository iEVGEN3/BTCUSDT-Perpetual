# Результаты исследований: Технические открытия и ограничения

## Проблема с верификацией карты на Koyeb
- **Наблюдение**: При регистрации на платформе Koyeb.com запрашиваются данные банковской карты для активации бесплатного тарифа.
- **Решение**: Koyeb не подходит для бесплатного использования без ввода карты. Проект переведен на Hugging Face Spaces + Neon.tech.

## Альтернативы хостинга без привязки карт

### 1. PythonAnywhere
- **Описание**: Облачный хостинг для Python-приложений. Бесплатный тариф не требует карты.
- **Ограничения**: Доступны только запросы из белого списка. Запуск фоновых процессов («Always-On») на бесплатном тарифе ограничен.

### 2. Hugging Face Spaces (Docker-контейнер)
- **Описание**: Платформа для развертывания веб-приложений и Docker-контейнеров. Бесплатный тариф CPU не требует карты.
- **Плюсы**: Позволяет боту работать непрерывно (always-on).
- **Минусы**: Файловая система временная (данные стираются при перезапусках контейнера).

## Базы данных без привязки карт
- **Neon.tech (Serverless PostgreSQL)**: Предоставляет полноценную бесплатную базу данных PostgreSQL без необходимости привязывать карту. Идеально подходит для сохранения подписок бота.

## Особенности работы с внешними API

### 1. Bybit REST API
- **Эндпоинт**: `https://api.bybit.com/v5/market/tickers?category=linear&symbol=<SYMBOL>`
- **Доступ**: Публичный эндпоинт, ключи авторизации для чтения котировок не требуются.

### 2. Технический анализ TradingView (библиотека `tradingview-ta`)
- **Библиотека**: `tradingview-ta`
- **Описание**: Получает сводные рекомендации (BUY, SELL, NEUTRAL) и точные значения индикаторов (RSI, MA) напрямую через публичный сканер TradingView без использования API-ключей.

## Ошибки интерфейса и обработки сообщений
- **Наблюдение**: Переход к бесшовному обновлению сообщений (Clean Screen) привел к исключениям `NameError: name 'edit_rich_message' is not defined` в `features/bot.py`.
- **Решение**: Добавлена функция `edit_rich_message`, использующая метод `bot.edit_message_text` с HTML-парсингом и безопасным откатом к тексту без HTML-тегов при возникновении ошибок разметки.
