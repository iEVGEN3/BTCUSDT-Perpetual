# 🦊 Телеграм-бот: Осторожный фьючерсный тренер

Бот-помощник для торговли фьючерсами на криптовалюту. Он написан на Python, получает котировки напрямую из Binance Futures/Bybit API, высчитывает уровни Стоп-лосс и Цели, объясняет сложную рыночную графику простыми словами с метафорами для новичков и присылает сигналы строго по заданному формату.

Этот бот идеально подходит для торговли внутри встроенного кошелька Telegram Wallet!

---

## 🛠️ Возможности бота

1. **Мгновенные сигналы по кнопкам**: запрашивайте сигналы для `BTC`, `ETH`, `SOL`, `TON`, `DOGE`, `NOT` прямо с интерактивной клавиатуры.
2. **Точный математический расчет**:
   - Стоп-лосс рассчитывается ровно на **1.5%** в противоположную сторону.
   - Цель 1: **+3.0%** в сторону сделки.
   - Цель 2: **+6.0%** в сторону сделки.
3. **Безопасность депозита**: бот очень осторожен. При противоречиях в индикаторах он выдает сигнал **ЧЕКАЙ**, чтобы защитить ваши средства.
4. **Умный подбор альтернатив**: если текущий актив находится в штиле (сигнал **ЧЕКАЙ**), бот автоматически проверяет другие основные монеты и предлагает сильную альтернативу, либо честно пишет: *«Сейчас по всем основным монетам лучше ждать.»*.
5. **Автоматическая рассылка (Алерты)**: пользователи могут подписаться на рассылку, и бот будет автоматически отправлять новые сильные сигналы на покупку или продажу раз в 15 минут в случае изменения тренда.
6. **База данных SQLite**: хранит информацию о подписках пользователей и историю сигналов для предотвращения дублирования сообщений.

* [`bot.py`](file:///f:/Antigravity/bot.py) — основной скрипт бота с обработчиками кнопок и фоновым планировщиком алертов.

## 🚀 Быстрый запуск

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python (версии 3.8 или выше). Выполните команду для установки библиотек:

```bash
pip install -r requirements.txt
```

### 2. Настройка токена
Откройте файл [`.env`](file:///f:/Antigravity/.env) и убедитесь, что там указан ваш токен от `@BotFather`:

```ini
TELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
```

### 3. Запуск бота
Запустите основной файл бота:

```bash
python bot.py
```

После запуска бот создаст базу данных `db.sqlite3` и выведет в консоль: `Бот успешно запущен и готов к работе...`.
